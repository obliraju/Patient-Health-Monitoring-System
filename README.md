Patient Health Monitoring System (PHMS)

A secure Android application for managing patient records and monitoring health information. The system provides authorized staff login, patient registration, digital medical records, and an intuitive dashboard for accessing patient details. It is designed with a scalable architecture to support future integration with IoT-based health monitoring devices and AI-powered healthcare features.

 Features

* Secure staff registration and login
* Patient registration and management
* Unique Patient ID generation
* Digital medical records
* Medical history tracking
* Ongoing medication records
* Doctor and hospital information
* Search and view patient details
* Room Database for offline storage
* Modern Material Design 3 UI
* MVVM architecture

Tech Stack

* Kotlin
* Jetpack Compose
* MVVM
* Room Database
* Material Design 3
* Navigation Component
* Coroutines

 Future Scope

* ESP32/Arduino IoT integration
* Real-time vital sign monitoring
* Firebase cloud synchronization
* PDF report generation
* QR code-based patient identification
* AI-based health analysis
* Appointment and medication reminders
